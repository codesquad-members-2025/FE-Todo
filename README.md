# FE-Todo

## 개발 접근 방식
본 프로젝트는 컴포넌트 기반 접근 방식(Component-Based Approach)으로 개발을 진행합니다.
- 각 기능 단위를 컴포넌트로 분리하여 개발
- HTML, CSS, JavaScript를 컴포넌트 단위로 통합 개발
- 기능별 독립적인 개발과 테스트 용이
